# Dream to Control: Learning Behaviors by Latent Imagination

## Summary
Dreamer (DreamerV1) introduces a reinforcement learning agent that learns long-horizon behaviors purely by imagining trajectories in the compact latent space of a learned world model. Rather than planning online at test time (as in PlaNet), Dreamer learns an actor-critic policy entirely within the latent imagination of a Recurrent State-Space Model (RSSM), using analytic gradients of learned state values propagated back through imagined trajectories. This makes the approach both more computationally efficient and more performant than prior model-based and model-free methods.

The agent operates in three stages: (1) learning a world model from experience via variational inference, (2) learning actor and critic networks by imagining trajectories in the latent space of the world model, and (3) executing the learned actor in the environment to collect new experience. On 20 challenging continuous control tasks from the [[DeepMind Control Suite]] with pixel observations, Dreamer achieves an average score of 823, exceeding the model-free D4PG agent (786) that trains on proprioceptive state, the prior model-based PlaNet agent (333), and the model-free A3C agent (244). Dreamer also achieves this performance with substantially less computation time: approximately 3 hours per 1M environment steps, compared to 11 hours for PlaNet and 24 hours for D4PG.

## Key Contributions
- First model-based RL agent to learn behaviors by backpropagating analytic value gradients through imagined latent trajectories
- Demonstrates that latent imagination with a learned world model can surpass both model-free and prior model-based methods in data efficiency, computation time, and final performance
- Introduces a three-stage training loop: world model learning, behavior learning by imagination, and environment interaction
- Achieves state-of-the-art results on 20 continuous control tasks from pixels using the [[DeepMind Control Suite]]

## Architecture / Method
Dreamer builds on the Recurrent State-Space Model (RSSM) introduced in PlaNet. The world model consists of several components:

**World Model (RSSM):**
- **Recurrent model:** h_t = f_phi(h_{t-1}, s_{t-1}, a_{t-1})
- **Representation model:** s_t ~ q_phi(s_t | h_t, o_t)
- **Transition model:** s_t ~ p_phi(s_t | h_t)
- **Observation model:** o_t ~ p_phi(o_t | h_t, s_t)
- **Reward model:** r_t ~ p_phi(r_t | h_t, s_t)

The world model is trained by optimizing the variational evidence lower bound (ELBO), which involves reconstruction of observations and rewards plus a KL divergence term between the representation and transition models.

**Behavior Learning by Latent Imagination:**
- An actor network pi_psi(a_t | s_t) outputs actions given latent states
- A critic network v_xi(s_t) estimates state values
- Imagined trajectories of length H are generated by iterating the transition model and actor
- The actor is optimized to maximize the sum of predicted rewards and estimated values along imagined trajectories using backpropagation through the dynamics (analytic gradients)
- The critic is trained via temporal difference learning on imagined trajectories

**Key distinction from PlaNet:** Instead of online planning with CEM (Cross-Entropy Method) at each step, Dreamer amortizes planning into a parametric policy trained via imagination, making it substantially faster at test time.

Dreamer substantially outperforms all baselines on average. Notably, Dreamer from pixels (823) surpasses D4PG from pixels (786) and even approaches D4PG performance on many tasks despite D4PG using proprioceptive features on some tasks. PlaNet, which uses the same world model but relies on online planning, achieves only 333 on average. On particularly challenging tasks like Acrobot Swingup, Hopper Hop, and Pendulum Swingup, Dreamer shows dramatic improvements over PlaNet, highlighting the value of learned behaviors over online planning.

Dreamer is approximately 3.7x faster than PlaNet and 8x faster than D4PG, while achieving superior final performance. This efficiency comes from amortizing planning into a parametric policy rather than running online optimization at each step.

## Metrics Used
- [[Episode Return]] — cumulative reward over an episode, averaged over 10 evaluation episodes per task
- Average Score — mean episode return across all 20 tasks, the primary aggregate metric
- Training Computation Time — wall-clock time per 1M environment steps, measuring computational efficiency
- [[Data Efficiency]] — performance as a function of environment steps, comparing learning speed

## Datasets Used
- [[DeepMind Control Suite]] — 20 continuous control tasks with pixel observations (64x64 RGB), including domains such as cartpole, cheetah, walker, finger, hopper, reacher, pendulum, cup, acrobot, and quadruped

## Related Papers
- PlaNet — predecessor model-based agent using the same RSSM world model but with online CEM planning instead of learned behaviors
- [[DreamerV2]] — successor work introducing discrete world models and achieving human-level Atari performance
- [[DreamerV3]] — further successor achieving general performance across diverse domains with a single configuration
- D4PG — model-free distributed actor-critic baseline
- A3C — model-free asynchronous advantage actor-critic baseline
