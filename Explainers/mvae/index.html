Interactive Explorations

Character Controllers
Using Motion VAEs

A collection of interactive visualizations exploring how a conditional VAE learns a structured action space for character animation — from latent space geometry to mixture-of-experts decoding and scheduled sampling.

Based on Ling, Zinno, Cheng & van de Panne, SIGGRAPH 2020

The Motion VAE (MVAE) learns to generate realistic, diverse human movement from motion capture data. Its core insight is that the latent variables of a trained VAE define a compact action space for character controllers. An RL agent (PPO) then selects latent codes to steer the character toward goals—reaching targets, following paths, or navigating mazes.

Each exploration below isolates one key mechanism from the MVAE framework. Drag through the latent space, toggle mixture-of-experts components, and scrub through the training curriculum to build intuition for how each piece contributes.
