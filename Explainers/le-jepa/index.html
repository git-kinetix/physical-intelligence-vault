Understanding Le-JEPA's
Self-Supervised Learning

A collection of interactive visualizations exploring Le-JEPA — a provably stable, scalable approach to self-supervised learning that replaces heuristics with a single principled regularizer.

Based on Balestriero & LeCun (2025)  ·  Code

Le-JEPA provides the first comprehensive theoretical framework for Joint-Embedding Predictive Architectures. It identifies the isotropic Gaussian as the optimal target distribution for JEPA embeddings and introduces SIGReg (Sketched Isotropic Gaussian Regularization) to enforce it.

The result is a method so simple it fits in approximately 50 lines of code, requires only one hyperparameter, and trains stably from 22M to 1.8B parameters without any tuning. Each visualization below isolates one key idea.

L = Lpred + λ · LSIGReg    λ ≈ 0.5
