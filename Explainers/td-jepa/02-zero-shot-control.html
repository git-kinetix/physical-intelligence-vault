Zero-Shot Control

Given a pre-trained TD-JEPA representation, an agent can navigate to any goal position without additional training. Click a goal and watch it plan in representation space.

Policy Selection in Latent Space

TD-JEPA pre-trains a bank of policies {pi_k}, each operating entirely in latent space. At test time, when a new reward function is specified (for example, “reach this goal position”), the system evaluates each policy’s successor features against the reward and selects the one that maximizes expected return — all without a single gradient step.

The interactive environment below simulates this process. The colored background shows the value landscape that TD-JEPA’s representation implies for each goal. Click anywhere to set a goal, and the agent will select the best policy and navigate there.

Notice how TD-JEPA’s agent takes smooth, purposeful trajectories, while random features produce erratic, wandering paths. The representation quality slider lets you simulate what happens when training is cut short: as quality drops, the agent’s trajectory becomes increasingly noisy and indirect, eventually approaching the random baseline.

In the paper, zero-shot control is evaluated on 65 tasks spanning locomotion, navigation, and manipulation. TD-JEPA achieves the highest average normalized return (628.8 on DMC RGB), beating BYOL-gamma (582.4) with the lowest variance.

The Value Landscape

A key advantage of TD-JEPA is that its representations encode value information implicitly. Below, the value landscape is visualized for multiple goals simultaneously. Bright regions have high predicted value (close to a goal); dark regions have low value (far from any goal, or blocked by obstacles).

Click to place up to four goals and see how the value landscape reshapes. The landscape is computed from the TD-JEPA successor features — brighter pixels indicate states whose latent representation is more aligned with the goal encoding.

The value landscape demonstrates a property that is impossible with one-step prediction: long-range spatial reasoning. Notice how the value field wraps around obstacles, creating natural corridors that guide the agent. A standard JEPA would produce a gradient that points straight toward the goal, ignoring barriers entirely.

The Policy Bank

TD-JEPA maintains a bank of K policies, each trained to optimize a different direction in latent space. At test time, the system evaluates each policy against the specified reward and selects the best one. The visualization below shows six pre-trained policies, each with a characteristic “specialty.”

Click a goal to see which policy gets selected (highlighted in green) and why. The bar chart shows each policy’s predicted return for the current goal.

This mechanism is what makes TD-JEPA truly zero-shot: no gradient steps are needed at test time. The policy bank was trained during the unsupervised phase on reward-free data, and each policy implicitly learned to navigate in a different direction through the environment. Policy selection reduces to a simple inner product between successor features and the reward vector — a computation that takes microseconds.

The theoretical result shows that TD-JEPA’s predictor recovers successor features in latent space. This means policy selection via inner product is optimal for the class of linear reward functions in the learned representation.
