Understanding TD-JEPA's
Value-Aware Representations

A collection of interactive visualizations exploring how temporal difference learning transforms JEPA representations into value-aware features for zero-shot reinforcement learning.

Based on Bagatella, Pirotta, Touati, Lazaric & Tirinzoni (2025)

Standard JEPA architectures predict the next latent state one or a few steps ahead. This captures local dynamics but misses the long-horizon consequences that reinforcement learning requires. TD-JEPA replaces the regression target with a temporal difference bootstrap, learning representations that encode not just what comes next, but what happens far into the future under different policies.

The result is a system that can solve new tasks zero-shot — given an arbitrary reward function at test time, TD-JEPA selects the policy whose latent successor features best align with that reward, without any fine-tuning. Evaluated on 65 tasks across 13 datasets, it matches or outperforms all baselines.

Core Objective LTD = || fk(φ(s)) − (ψ(s′) + γ · sg(fk(φ(s′)))) ||2

Architecture at a Glance

TD-JEPA consists of four tightly coupled components that operate entirely in latent space. The state encoder compresses observations, the task encoder defines reward-like signals, the predictor learns discounted future features, and a bank of policies enables zero-shot selection at test time.

φ

State Encoder

Maps observations (pixels or proprioception) to latent state representations z = φ(s). Shared across all policies.

ψ

Task Encoder

Maps observations to task-relevant features w = ψ(s). Defines the reward signal in latent space for TD targets.

fk

Predictor

Policy-conditioned predictor that learns successor features. Trained with TD loss using γ-discounted bootstrap targets.

πk

Policy Bank

Set of K policies operating in latent space. At test time, select the policy maximizing alignment with reward.

The key theoretical insight is that the predictor converges to successor features — the expected discounted sum of future task encodings under each policy. This enables zero-shot evaluation of any linear reward in the representation space.

With proper initialization, TD-JEPA provably avoids representation collapse. The learned encoders capture a low-rank factorization of the long-term policy-conditioned transition dynamics.

How It Differs from Standard JEPA

Standard JEPA uses a one-step regression loss: the predictor is trained to match the encoder output for the very next observation. This captures local transition dynamics but nothing about long-term consequences.

Standard JEPA LJEPA = || f(φ(s)) − φ(s′) ||2

TD-JEPA makes two critical modifications. First, it introduces a separate task encoder ψ for the target, decoupling what gets predicted from what the state representation encodes. Second, it adds a discounted bootstrap term γ · sg(fk(φ(s′))) that recursively incorporates future predictions into the target.

TD-JEPA LTD = || fk(φ(s)) − (ψ(s′) + γ · sg(fk(φ(s′)))) ||2

Unrolling the recursion reveals the predictor is learning the discounted sum of all future task encodings: ψ(s1) + γψ(s2) + γ2ψ(s3) + …  — precisely the successor features under policy k. At γ = 0.99 (the value used in all experiments), the effective horizon is roughly 100 steps.

Key Results

TD-JEPA is evaluated on 65 tasks across 13 datasets spanning locomotion, navigation, and manipulation. The chart below shows performance on the DMC benchmark, where TD-JEPA achieves the highest average normalized returns in both the RGB (pixel) and proprioceptive settings.

On OGBench, TD-JEPA (41.34% success rate, RGB) is competitive with the best baseline BYOL-γ (41.58%) while maintaining the lowest variance (0.45 vs 0.64). The consistency across benchmarks and modalities demonstrates the generality of the TD-JEPA approach.

A particularly notable result: TD-JEPA’s performance drop from proprioception to vision is only 32.4 points (661.2 to 628.8), compared to 192 for FB and 229 for HILP. The TD objective serves as an implicit curriculum for the visual encoder.
