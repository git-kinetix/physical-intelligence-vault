LeWM Internals · Part 4

Causal Self-Attention

How the ARPredictor looks backward in time — never forward — to predict the next frame embedding from a window of past observations.

The ARPredictor processes frame embeddings through 6 layers of self-attention. But unlike standard transformers in language modeling, here the "tokens" are entire frame embeddings, and the "sequence" is a temporal window of 3–4 frames. Causal masking ensures predictions can only look backward in time.

Attention Matrix Explorer

Each attention head learns to focus on different temporal relationships. Some heads attend primarily to the most recent frame, others anchor on the first frame in the context window, and some distribute attention uniformly. Together, the 16 heads capture a rich set of temporal patterns.

Why Causal?

During training the model sees the full sequence, but it must learn to predict each frame using only past context. If it could peek at future frames, the loss would drop but the model would be useless at inference time when future frames don't exist yet. The causal mask prevents this information leak.

Q · K · V Computation

Scaled dot-product attention computes compatibility scores between queries and keys, then uses those scores to weight a mixture of values. The scaling factor 1/√dhead prevents the softmax from saturating when the dimension is large.

Multi-Head Attention

In LeWM's ARPredictor, attention operates differently from a standard transformer block: the ConditionalBlock uses AdaLN-zero, where the action embedding modulates the normalization parameters before attention is applied. This means the attention pattern itself is influenced by the action taken.
