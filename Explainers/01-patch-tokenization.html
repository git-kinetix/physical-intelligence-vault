Exploration 01

From Pixels to Tokens

How the Vision Transformer encoder in LeWM converts a 224×224 pixel frame into a single 192-dimensional CLS embedding — the compact representation that the world model learns to predict.

The Vision Transformer sees the world through patches. Rather than processing pixels individually, it divides each frame into a regular grid of 14×14-pixel patches — 256 in total — and treats each as a token. This is a fundamentally different inductive bias from convolutional networks: the ViT makes no assumption about spatial locality until the self-attention layers learn it from data.

Patch Grid Explorer

 

Each patch is linearly projected into the transformer's hidden space via a learned embedding matrix. A special classification token (CLS) is prepended to the sequence, serving as the aggregate representation of the entire image. Learned positional encodings are added so the model knows where each patch originated in the grid.

ViT-tiny uses a hidden dimension of 192 and 3 attention heads per layer, making it lightweight enough for real-time robotics inference while retaining strong representation capacity.

Token Sequence

257 TOKENS = 1 CLS + 256 PATCHES
 

The token sequence passes through 12 self-attention layers. Each layer allows every token to attend to every other, building up increasingly abstract representations. But for LeWM, only the CLS token matters — it becomes the frame's embedding.

# From jepa.py — encoding a frame output = self.encoder(pixels, interpolate_pos_encoding=True) pixels_emb = output.last_hidden_state[:, 0] # CLS token only emb = self.projector(pixels_emb)

ViT Processing Flow

The CLS token is then projected through a 2-layer MLP with BatchNorm and GELU activation, reducing it from the ViT's hidden dimension to the 192-dimensional embedding space where prediction happens.

The MLP projector architecture: Linear(192 → 2048) → BatchNorm1d(2048) → GELU → Linear(2048 → 192). The wide hidden layer allows the network to learn a nonlinear mapping from the ViT's representation space to the prediction space.

CLS → Projector → Embedding

Summary

From a raw 224×224 pixel frame, the ViT-tiny encoder produces a single 192-dimensional vector that captures the visual state of the environment. This compact embedding is what the world model's autoregressive predictor learns to forecast forward in time, conditioned on actions.
