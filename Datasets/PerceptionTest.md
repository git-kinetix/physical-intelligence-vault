# PerceptionTest

## Description
The Perception Test is a multimodal diagnostic benchmark from Google DeepMind for comprehensively evaluating the perception and reasoning skills of video models. It features 11,600 real-world videos (average 23 seconds each) filmed by ~100 participants worldwide, with tasks spanning object tracking, point tracking, action localization, sound localization, and video question-answering.

## Format
Real-world videos with multi-task annotations including temporal and spatial labels, multiple-choice QA, and tracking ground truth.

## Size
11,600 videos, average 23 seconds each.
