# Panda-70M

## Description
Panda-70M is a large-scale video-caption dataset created by Snap Research, containing 70 million high-quality video-caption pairs. Captions are generated using multiple cross-modality teacher models. The videos are curated from the publicly available HD-VILA-100M dataset, with a focus on high-quality, semantically rich video-text alignment for training video generation and understanding models.

## Format
Video clips paired with text captions. Distributed as CSV files listing video metadata and captions, with scripts provided to download the source videos.

## Size
70 million video-caption pairs, curated from 3.8 million high-resolution source videos.
