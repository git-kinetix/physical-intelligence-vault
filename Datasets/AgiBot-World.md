# AgiBot-World

## Description
AgiBot-World is the first large-scale robotic learning dataset designed to advance multi-purpose humanoid robotic policies. The Beta release contains over 1 million trajectories from 100 robots with a total duration of 2,976 hours, making it one of the largest real-world robot manipulation datasets available.

## Format
Trajectory data with RGB images, depth, proprioceptive states, and actions from multiple robot embodiments. Available in LeRobot-compatible format.

## Size
1,003,672 trajectories (~43.8 TB), 2,976 hours of data from 100 robots (Beta release). Alpha release: 100,000+ trajectories, 300 hours.
