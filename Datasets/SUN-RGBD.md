# SUN-RGBD

## Description
SUN-RGBD is a comprehensive RGB-D scene understanding benchmark suite containing 10,000 RGB-D images captured by four different depth sensors (Intel RealSense, Asus Xtion, Kinect v1, Kinect v2). It provides dense annotations with 146,617 2D polygons, 58,657 3D bounding boxes with accurate orientations, 3D room layouts, and semantic category labels, at a scale comparable to PASCAL VOC.

## Format
RGB-D images with 2D semantic segmentation polygons, 3D bounding boxes, room layout annotations, and object category labels.

## Size
10,000 RGB-D images, 146,617 2D polygons, 58,657 3D bounding boxes.
