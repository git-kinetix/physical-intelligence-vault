# PASCAL VOC12

## Description
PASCAL VOC 2012 (Visual Object Classes Challenge 2012) is a foundational benchmark for object detection, semantic segmentation, and image classification. It provides annotations for 20 object classes plus background, and has served as one of the earliest standardized evaluation benchmarks for these tasks. Despite its modest size, it remains widely used as a transfer learning evaluation target.

## Format
JPEG images with XML annotation files (PASCAL VOC format) for bounding boxes, and PNG segmentation masks for semantic/instance segmentation.

## Size
7,282 images with 19,694 labeled objects across 21 classes. Splits: 1,464 train, 1,449 val, 1,456 test, and 2,913 trainval. Approximately 2 GB total.
