# Assembly-101

## Description
Assembly-101 is a large-scale multi-view video dataset for understanding procedural activities, featuring participants assembling and disassembling 101 different toys. It provides recordings from 8 static and 4 egocentric viewpoints, with 4,321 sequences totaling 513 hours of footage, annotated with over 100K coarse and 1M fine-grained action segments plus 18M 3D hand poses.

## Format
Multi-view video (8 fixed + 4 egocentric cameras) with coarse and fine-grained temporal action annotations, 3D hand pose data, and object interaction labels.

## Size
4,321 sequences, 513 hours, 100K+ coarse action segments, 1M+ fine-grained segments, 18M 3D hand poses.
