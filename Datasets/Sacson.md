# Sacson

## Description
SACSoN (Scalable Autonomous Control for Social Navigation) is the largest visual navigation dataset of autonomous robots operating in human-occupied spaces, spanning over 75 hours and 4,000 rich interactions with humans. It provides data for training socially-aware robot navigation policies in real-world pedestrian environments.

## Format
Visual navigation trajectories with RGB images, odometry, and human interaction annotations.

## Size
75+ hours of navigation data, 4,000+ human interactions.
