# TOMATO

## Description
TOMATO is a temporal reasoning benchmark published at ICLR 2025 (from Yale NLP) for assessing visual temporal reasoning capabilities in multimodal foundation models. It includes 1,484 human-annotated multiple-choice questions across six temporal reasoning tasks: rotation, direction, velocity/frequency, shape/trend, visual cues, and action counting, using 1,417 curated videos from YouTube, existing datasets, and original content. The best model (Qwen2-VL-72B) achieves only 37.9% vs. 95.2% human accuracy.

## Format
Video-based multiple-choice QA benchmark. Videos from three sources: YouTube (398), existing datasets (214), and original recordings (805).

## Size
1,484 questions, 1,417 videos, 6 temporal reasoning tasks.
