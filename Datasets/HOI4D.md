# HOI4D

## Description
HOI4D is a large-scale 4D egocentric dataset for category-level human-object interaction, containing 2.4 million RGB-D egocentric video frames across 4,000 sequences. It features 9 participants interacting with 800 object instances from 16 categories in 610 indoor rooms, with rich annotations including object poses, hand poses, and interaction semantics.

## Format
RGB-D egocentric video with 3D object pose annotations, hand pose, action segmentation labels, and panoptic segmentation at the category level.

## Size
2.4M RGB-D frames, 4,000 sequences, 800 object instances, 16 categories, 610 indoor rooms.
