# Cityscapes

## Description
Cityscapes is a large-scale dataset for semantic understanding of urban street scenes, featuring high-resolution images captured from a vehicle driving through 50 different cities across Germany and neighboring countries. It provides dense pixel-level annotations for semantic segmentation, instance segmentation, and panoptic segmentation, serving as a key benchmark for autonomous driving perception.

## Format
High-resolution stereo images (2048x1024 pixels) with fine-grained pixel-level annotations in PNG format. Includes 30 semantic classes grouped into 8 categories.

## Size
5,000 finely annotated images (2,975 training, 500 validation, 1,525 test) plus 20,000 coarsely annotated images. Approximately 11 GB for images, 241 MB for annotations.
