# RECON

## Description
RECON (Rapid Exploration for Open-World Navigation) is a large-scale robot navigation dataset containing over 5,000 self-supervised trajectories collected across 9 distinct real-world environments. It provides multimodal sensor data for learning navigation policies that can rapidly explore and navigate to novel goals in open-world settings.

## Format
Multimodal trajectory data including stereo RGB camera images, thermal camera, 2D LiDAR, GPS, and IMU measurements.

## Size
~50 GB download; 5,000+ trajectories across 9 environments.
