# MS-COCO 2017

## Description
MS-COCO (Microsoft Common Objects in Context) 2017 is a large-scale object detection, segmentation, and captioning dataset. It features images of complex everyday scenes containing common objects in their natural context, with dense annotations including bounding boxes, segmentation masks, keypoints, and captions.

## Format
JPEG images with JSON annotation files (COCO format). Annotations include object detection bounding boxes, instance segmentation masks, keypoint annotations, and image captions.

## Size
118K training images, 5K validation images, and 20K test images. Approximately 19.6 GB total download.
