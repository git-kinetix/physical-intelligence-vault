# DexMimicGen

## Description
DexMimicGen is an automated data generation system that synthesizes large-scale trajectories for humanoid robots with dexterous hands from a small number of human demonstrations. From just 60 source human demonstrations, it autonomously generates over 20,000 demonstrations for bimanual dexterous robots across 9 tasks, multiple simulators, and real-world settings.

## Format
Trajectory data with robot states, actions, and observations for bimanual dexterous manipulation in simulation (Isaac Sim, Robosuite) and real-world settings.

## Size
20,000+ generated demonstrations from 60 source demonstrations across 9 tasks.
